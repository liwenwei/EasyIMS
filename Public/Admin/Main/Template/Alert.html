button:null //按钮组 （数组调用方法同上）

在以上框框中输入值点下边按钮传值

给异步页面赋值 
$('.obj').click(function () {
if (test) {
$.dialog({
id: 'test',
content: 'url:Template/test1.html',
data: $('#test').val()
})
}
})
 if(test)  判断窗口是否打开，必须指定窗口Id
data：为传递参数
